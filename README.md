深圳大学选课助手

用于 Chrome / Edge 的选课辅助插件，帮助整理深圳大学选课页面中的课程信息，并生成不同的排课方案。

## 安装

1. 下载并解压本项目。
2. 打开 Chrome 的 `chrome://extensions`，或 Edge 的 `edge://extensions`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择 `course-scraper-extension` 文件夹。

## 可以做什么

- 读取课程大类，再选择需要爬取的课程。
- 自动读取教师、周数、节次、时间和学分。
- 自动翻页读取多页课程。
- 导出 CSV 或 JSON。
- 按上课天数、课程数量、学分、早八、晚课和课程密度生成排课方案。
- 在网页左侧预览课表，同时在右侧调整排课方案。

## 提醒

请以教务系统最终显示的课程与选课结果为准。本工具只用于展示不同课程组合与排课方案，不会自动提交或锁定选课结果。

选课不是一次随手点击，是在决定下学期的幸福。  
课表不是一张普通表格，是你每天要面对的生活。
