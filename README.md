# 选课助手

一个用于 Chrome / Edge 的 Manifest V3 浏览器扩展。它可以从教务系统页面读取课程大类与教学班信息，导出 AI 友好的结构化课程数据，并根据时间冲突生成可选课表方案。

当前针对深圳大学选课页面的 `cv-row` 课程结构和校公选课列表做了适配，同时保留了对常见表格、课程卡片和课表网格的通用解析。

> 本工具只在浏览器本地处理当前页面内容，不会替你提交选课或修改教务系统数据。请以教务系统最终显示的结果为准。

## 功能

### 两阶段课程爬取

- 先读取课程大类，再由用户勾选需要的课程。
- 自动模拟点击课程大类，读取不同教师、时间和地点的教学班。
- 支持读取当前页，或自动翻页读取课程大类和已选课程。
- 跨页爬取会累积已读结果；某一页没有已选大类时会自动跳过，不会清空前面已读取的课程。
- 完成自动翻页后，自动返回第一页并收起已展开的课程详情。
- 支持设置最大翻页数、翻页间隔和课程名称正则筛选。默认翻页间隔为 0.5 秒。

### 校公选课与通用页面解析

- 自动识别校公选课列表：课程信息已在表格中完整展示时，直接读取，不再错误尝试展开详情。
- 支持常见 HTML 表格、课程列表、课程卡片、周课表网格、框选区域和选中文字解析。
- 同一课程的完全相同上课时间会合并，并汇总教师信息。

### 导出与缓存

- 导出 CSV 或 JSON，字段包含课程名称、节数、学分、教师、周数、时间、课程号、类别、性质与原始文本。
- 可切换为“详细信息”或“课程名称 + 时间”的精简导出。
- 缓存按网站域名保存；关闭扩展弹窗不会清空数据。
- 已生成的排课方案、当前选中方案和排课设置会按网站保存；重新打开扩展时自动恢复。
- 可单独清空当前网站的课程大类、课程、排课方案、删除记录与筛选设置。

### 智能排课

- 一门课程只选择一个教学班；自动识别时间冲突。
- 冲突时可以跳过课程，尽量保留更多可选课程。
- 支持“最少上课天数”“尽量无 1-2 节课”“最多课程数”等简单策略。
- 支持高级自定义：上课天数、是否接受早八、是否接受晚课、是否接受 11-14 节课程。
- 方案综合考虑上课天数、课程数、学分、早八、晚课和日内课程密度；同等情况下优先总学分更高的方案。
- 显示每个方案的总学分、无课工作日、冲突跳过数量与综合评分。
- 可在原网页左侧停靠预览当前排课方案的周课表，右侧插件保持可用；切换方案时预览会同步更新。

### 课程筛选

可以在生成方案前排除下列课程，并可一键恢复：

- 公共选修课
- 公共必修课
- 专业选修课
- 专业必修课
- 体育类课程
- MOOC 类课程

## 安装

1. 下载或克隆本仓库。
2. 打开 Chrome 的 `chrome://extensions`，或 Edge 的 `edge://extensions`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本仓库中的 `course-scraper-extension` 文件夹。
6. 打开教务系统的选课页面，点击浏览器工具栏中的“选课助手”。

每次更新本地代码后，请在扩展管理页点击“重新加载”，再刷新教务系统页面。

## 使用流程

### 读取并爬取课程

1. 默认不会自动爬取。打开选课页面后，点击“读取本页大类”，或使用“读取全部页面”。
2. 在课程大类列表中勾选需要的课程；可使用全选和取消全选。
3. 点击“爬取本页已选”读取当前页，或点击“爬取全部已选”自动翻页读取所有页面中的已选课程。
4. 爬取完成后，在课程表格中查看、删除不需要的教学班，或使用课程类别筛选。

### 生成排课方案

1. 在“智能排课”中选择简单排课或高级自定义排课。
2. 选择策略或设置限制条件。
3. 点击“生成方案”。
4. 从方案下拉列表中切换不同结果，查看总学分、无课工作日和评分。
5. 使用“网页预览排课方案”查看周课表，或导出当前方案。

## 项目结构

```text
course-scraper-extension/
├── manifest.json          # 扩展配置
├── popup.html             # 扩展弹窗界面
├── popup.css              # 弹窗样式
├── popup.js               # 状态、导出与交互逻辑
├── content-script.js      # 页面读取、展开、翻页与预览逻辑
├── scheduler.js           # 排课与评分算法
└── scheduler.test.js      # 排课算法测试
```

## 本地测试

项目不依赖额外的 npm 包。安装 Node.js 后，在扩展目录执行：

```powershell
node --check popup.js
node --check content-script.js
node --check scheduler.js
node --test scheduler.test.js
```

## 隐私与权限

- `activeTab`：仅在你主动打开扩展时访问当前标签页。
- `storage`：保存本地课程缓存、筛选设置和排课结果。
- `scripting`：支持在当前页面进行框选和课程表预览。
- `host_permissions: <all_urls>`：用于兼容不同教务系统页面；扩展不会将课程数据上传到外部服务器。

## 已知限制

- 教务系统页面结构变化后，课程大类、详情入口或分页控件可能需要重新适配。
- 某些课程的时间文本不规范时，可能无法参与自动排课，但仍会保留在导出结果中。
- 翻页过快可能触发教务系统会话限制；若遇到登录失效，请重新登录并适当增大翻页间隔。
- 自动生成的方案仅按时间冲突和当前设置筛选，不代表满足培养方案、先修课、容量或选课优先级要求。
